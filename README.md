# AVRModPlay

This is a header-only Arduino library for playing Amiga Protracker songs in MOD format.

Optimized for the best performance for 16MHz AVR Arduino boards.

## Limitations

AVRModPlay only supports 4-voice MODs (tags: `M.K.`, `FLT4`, `4CHN`).

### Supported Commands

+---------+------------------------+----------+----------------------------+

| Axy     | Volume Slide           | Full     |                            |
+---------+------------------------+----------+----------------------------+
| Bxy     | Jump To Pattern        | Full     |                            |
+---------+------------------------+----------+----------------------------+
| Cxx     | Set Volume             | Full     |                            |
+---------+------------------------+----------+----------------------------+
| Dxx     | Pattern Break          | Full     |                            |
+---------+------------------------+----------+----------------------------+

+---------+------------------------+----------+----------------------------+

+---------+------------------------+----------+----------------------------+
